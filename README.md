Сегментация по устройствам

** Mobile:**
 
Доминирует по доле пользователей (68%).

Меньше страниц за сессию (5.13) → возможно, быстрый вход/выход.

Короткие сессии (37.2 мин) → менее глубокая вовлечённость.

Частота сессий (6.35) — высока, но ниже, чем у desktop.

Конверсия (10.34%) — чуть ниже desktop.

Средний чек (710.7) — самый низкий.

Поведение: частые, короткие визиты, низкий средний чек.


** Desktop:**

Меньше пользователей (29.5%), но более ценные.

Больше страниц (7.36) и дольше сессии (52.7 мин) → глубокое взаимодействие.

Высокая конверсия (11.79%) и AOV (875.18) → основной источник выручки.

ARPU = 136.43 — почти в 2 раза выше, чем у mobile!

Поведение: серьёзные действия, покупки, аналитика.
